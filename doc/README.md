#### Iniciando projeto Python
* conferir a versão
```
python --version
```

* cria uma venv
```
python -m venv venv
```

* ativando o env
```
venv\Scripts\activate
```

#### Instalar o django
```
pip install django
```

* testando django
```
django-admin
```

* criando projeto
```
django-admin startproject ecommerce
```

* entra na pasta
```
cd ecommerce
```

* rodar o projeto
```
python manage.py runserver
```

* analisar
python manage.py migrate

python manage.py createsuperuser
